## 📌 Overview

This project builds a complete HR analytics pipeline across 50,000+ employee records — from raw data cleaning in Excel, through SQL-based ETL and analysis, to interactive visualizations in Tableau and Power BI — to identify why employees leave and where the business should focus retention efforts.

## ✨ Key Insights & KPIs

- 📊 Tracked **10+ KPIs**: attrition rate, average hourly rate, tenure, performance rating
- 📉 Identified an **overall attrition rate of 50.21%**, with **R&D department highest at 51.21%**
- 🔍 Uncovered attrition drivers by:
  - Department
  - Income level
  - Marital status
  - Promotion timing
- 💡 Translated findings into actionable **retention recommendations**

## 🛠️ Tech Stack & Pipeline

| Stage | Tool | Purpose |
|---|---|---|
| 1. Data Cleaning | Excel | Initial cleanup & preprocessing |
| 2. ETL & Analysis | SQL (MySQL) | Joins, CTEs, window functions, stored procedures |
| 3. Visualization | Tableau | Interactive exploratory dashboards |
| 4. Reporting | Power BI | KPI dashboard & DAX-based reporting |

## 📁 Project Files

- `HR_Dashboard__PowerBI_.pbix` — Power BI dashboard
- `HR_Dashboard__Tableau_.twbx` — Tableau workbook
- `HR_Dashboard__Excel_.xlsx` — Excel data & analysis *(large file — see note below)*
- `HR_Analytics__Report_.pptx` — Presentation summary of findings
- `ETL_and_Joins.sql` — Database creation, data loading & table joins
- `Queries_and_Procedures.sql` — KPI queries, CTEs & stored procedures

> **Note:** The Excel file is large (~28MB). If not available directly in this repo, contact for the full file.

## 🚀 How to View

1. **Power BI:** Download `.pbix` and open in Power BI Desktop
2. **Tableau:** Download `.twbx` and open in Tableau Desktop/Reader
3. **SQL:** Run `ETL_and_Joins.sql` first to set up the database, then `Queries_and_Procedures.sql` for KPI queries
4. **Presentation:** Open `.pptx` for a quick summary of findings
